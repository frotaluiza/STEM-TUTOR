### Key RAG Features:
- **Hybrid retrieval**: BM25 + vector search fused via `QueryFusionRetriever` with reciprocal rank fusion
- **Chunking**: `SentenceSplitter` with configurable `chunk_size` and `chunk_overlap`
- **Index versioning**: Embedding signature-based version selection (rebuilds index when model changes)
- **Multi-query smart retrieval**: `SmartRetriever` generates query variants and aggregates results
- **Document support**: PDF (via PyMuPDF/pdfplumber/pypdf), DOCX, XLSX, PPTX, images (with LLM description), Markdown, text
- **Multiple embedding providers**: OpenAI, Gemini, OpenRouter, Jina, Cohere, Ollama, vLLM, SiliconFlow, Aliyun
