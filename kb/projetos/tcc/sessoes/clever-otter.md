### Frameworks de Malha Fechada (Closed Loop) para Testes com Agentes

- Planeja com base no diff/PR → Gera código executável → Executa em sandbox (Docker) → Auto-healing se falhar
- Ferramentas: [Tenet](https://github.com/JeiKeiLim/tenet), [Codeharness](https://github.com/iVintik/codeharness), [Ghost](https://github.com/tripathiji1312/ghost), [AgentForge](https://arxiv.org/pdf/2604.13120)

- **Feed-forward**: regras de arquitetura via `AGENTS.md` / `SKILL.md`
